# PenguinMod Desktop

An unofficial desktop version of [PenguinMod](https://penguinmod.com/)

## Finding logs

You can find the logs in the following places:

-   For Windows: `%localappdata%/com.penguinmod.app/logs`
-   For MacOS: `~/Library/Logs/com.penguinmod.app`
-   For Linux: `~/.local/share/com.penguinmod.app/logs`

## Deep Links

PenguinMod Desktop uses the `penguinmod` protocol for deep links.

### Projects

Deep link can open projects directly from penguinmod.\
These links follow the format `penguinmod:projects/<project_id>`

Additionally, you can open projects with a link to the project's file.\
These links follow the format `penguinmod:project/<file_url>`
